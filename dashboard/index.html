🌊 What is Oceanus?
Oceanus is a multi-agent RL environment where AI agents learn to recover ghost fishing nets from the ocean. 640,000 tonnes of ghost gear is lost annually — killing marine life for decades. This dashboard shows AI agents solving that problem in real-time.

There are 6 AI agents running simultaneously: 4 ASV drones physically cleaning nets on the ocean grid, and 2 policy agents negotiating a treaty to slow net spawning at the source. A chaos adversary injects storms, equipment failures, and policy crises to test the agents.

🎮 Button Reference
🚀 DEMO
Auto-switches to REPLAY tab and plays the pre-recorded trained agent episode. Shows the best learned behavior — 13 nets cleaned, treaty signed.
▶ START
Starts a fresh live simulation. Mock heuristic agents run in real-time. The 3D grid, KPIs, and event feed all update live via WebSocket.
■ STOP
Pauses the running simulation. The current state is preserved — click START again to resume from a new episode.
↺ RESET
Resets the simulation to a fresh state. Clears all charts, event log, and reward history. Starts a new episode from step 0.
📑 The Three Modes
LIVE Mode
Real-time simulation running on the server. 6 agents act every 80ms. Watch the 3D ocean grid, agent intelligence panel, policy negotiation, and live event feed update in real-time.
REPLAY Mode
Load pre-recorded episodes. Click ✅ Load Trained or ❌ Load Baseline. Use transport controls (⏮⏪▶⏩) to scrub through. Timeline markers show key events.
COMPARE Mode
Side-by-side analysis of baseline vs trained agent. Shows reward curves, biodiversity charts, and final grid state. Click Load Comparison Data to populate.
Speed Controls (Replay)
0.5× · 1× · 2× · 4× playback speed. Click any speed button while replay is playing to change pace instantly.
🗺️ Reading the 3D Ocean Grid
Grid Cells (20×20)
Each cell represents an ocean sector. Color = ghost net density:
■ Dark blue = clear water   ■ Cyan = low density
■ Orange = medium   ■ Red = high density
ASV Drones (4 agents)
● ASV-1   ● ASV-2   ● ASV-3   ● ASV-4
Drones smoothly lerp to new positions each step. Glowing rings pulse when active. Particle bursts appear when a net is cleaned.
Sectors (A1–D4)
The grid is divided into 16 named sectors (A1, A2... D4). Agents broadcast their sector position to coordinate. Labels are visible as floating sprites.
Camera Controls
Drag to rotate · Scroll to zoom · Auto-rotates when idle for 6 seconds. The camera slowly orbits the scene to show all angles.
📊 KPI Dashboard
🌱 Biodiversity Index (big card)
The primary metric. Starts at 100%. Drops as ghost nets accumulate (−0.3% per net). Recovers as nets are cleaned (+0.1% per clean). Target: ≥75% for full ecosystem recovery bonus (+100 reward).
🕸 Active Nets / ✅ Cleaned
Active = ghost nets currently drifting on the grid. Cleaned = total removed this episode. Nets spawn every 5 steps (High rate) or every 15 steps (Low rate, after treaty).
📜 Treaty Status
Shows the current policy negotiation state: No Mandate → Proposed → Active. When Active, net spawn rate drops from High to Low — directly helping ASV agents.
💰 Reward / ⚡ Chaos
Reward = cumulative RL score this episode. Chaos = number of adversary events fired (storms, trawlers, schema drift, equipment failures).
🤖 Agent Intelligence Panel
Each ASV card shows: battery level (depletes by 1 per move, red when <25%), last action (MOVE/CLEAN/BROADCAST/SCAN), reasoning text, current sector, position, and nets cleaned.

The Coordination Score bar rises when agents broadcast positions to each other (+15% per broadcast) and clean nets (+20% per clean). High coordination = agents working as a team.

The Policy Negotiation panel shows the treaty progress bar and timeline. When Port Authority proposes a treaty and Fleet Manager accepts, the spawn rate drops immediately.
🧠 The AI Training Behind This
The agents shown in DEMO mode were trained using GRPO (Group Relative Policy Optimization) — the same algorithm behind DeepSeek-R1. Model: Qwen2-0.5B-Instruct with 4-bit LoRA (2.1M trainable parameters), trained for 600 steps on Google Colab T4.

Before training: model outputs random text → reward −1.0 per step.
After training: model outputs {"intent": "clean"} when a net is visible → reward +8.0 per step.

Verifiable improvement: +9.0 reward points per step.
